Links in HTML (Anchor tag)

There are to types of linking in HTML

  1. Relative Linking: The type of linking in which we link our own page is called Relative Linking. For example: About us, Contact us, Confirm Order, etc
  2. Absolute Linking: The type of linking in which we link external web pages such as Google or Youtube.

Different Anchor Tag attributes:

  • target="_self": Default current tab will go to specified link

    • Go to Google

  • target="_blank": open the specified link in new tab

    • Go to Google in blank tab

  • download: download the specified relative link

    • Download video

  • download="Headings in HTML.html": download the specified relative link with the given name

    • Download video

Anchor tag href attribute:

  • href="whatsapp://923181236267": Opens whatsapp with the given number

    • Open Whatsapp (opens in mobile)

  • href="tel:923181236267": make a telephone call

    • Call the number

  • href="mailto:xorahsan@gmail.com": use to email the mentioned address

    • Email now

URL Scheme (protocols):

There are many IANA registered url schemes and we can also make our own. The current list of url scheme of different url scheme can be viewed on Wikipedia. Moreover, it also depends upon browser whether it allows that scheme or not!

Shift to different heading on current page using HTML only!:

By giving the particulr HTML element id in a link, we can shift to different section of page without using css or javascript!
